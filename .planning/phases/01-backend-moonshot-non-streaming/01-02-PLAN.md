  <action>
    Implement `async def send_openai_message(self, body: dict[str, Any], headers: dict[str, str]) -> BackendResponse`:

    1. Resolve the model alias:
       - `model = body.get("model", "")`
       - If model == `"kimi-latest"`, call `resolved = await self._resolve_model_alias(model)` and create a copy of `body` with `model` replaced by `resolved`. Do NOT mutate the original `body` dict.

    2. Build upstream headers:
       - Start with `{"content-type": "application/json"}`.
       - Add `Authorization: Bearer {self.api_key}`.
       - Do NOT forward `Authorization` from the client `headers`.
       - Optionally forward `x-request-id` or similar trace headers if present, but keep it minimal for v1.

    3. Make the request:
       - `response = await self._client.post(f"{self.base_url}/chat/completions", json=request_body, headers=upstream_headers)`
       - Set `timeout=self.timeout`.

    4. Build the `BackendResponse`:
       - If `response.status_code < 400`: return `BackendResponse(body=response.json(), status_code=response.status_code, headers={"content-type": "application/json"})`
       - Else: log the error with status and first 200 chars of response text, then return `BackendResponse(body=response.json(), status_code=response.status_code, headers={"content-type": "application/json"}, error=response.text)`

    5. Wrap the request in try/except for `httpx.HTTPError` and similar:
       - On transport error, return `BackendResponse(body={"error": {"message": str(e), "type": "api_error"}}, status_code=502, error=str(e))`

    6. Ensure `body` may contain `stream: false` or no `stream` key. Do NOT set `stream: true` here; that is Phase 4.
  </action>

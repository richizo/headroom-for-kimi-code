    1. Module docstring explaining this is the native Moonshot/Kimi backend.

    3. Import `Backend`, `BackendResponse`, `StreamEvent` from `.base`.
    4. Define `DEFAULT_MOONSHOT_BASE_URL = "https://api.moonshot.ai/v1"`.
    5. Define `DEFAULT_MOONSHOT_TIMEOUT = 60.0`.
    6. Implement class `MoonshotBackend(Backend)`:
       - `__init__(self, api_key: str | None = None, base_url: str = DEFAULT_MOONSHOT_BASE_URL, timeout: float = DEFAULT_MOONSHOT_TIMEOUT) -> None`
         - `self.api_key = api_key or os.environ.get("MOONSHOT_API_KEY")`
         - Raise `ValueError("MOONSHOT_API_KEY is required")` if key is still None.
         - `self.base_url = base_url.rstrip("/")`
         - `self.timeout = timeout`
         - `self._client = httpx.AsyncClient(timeout=timeout)`
         - `self._model_alias_cache: dict[str, str] = {}`
         - Log initialization with `logger.info("Moonshot backend initialized (base_url=%s)", self.base_url)`.
       - `name` property returns `"moonshot"`.
       - `map_model_id(self, model: str) -> str` returns the model as-is for now (alias resolution in Plan 02).
       - `supports_model(self, model: str) -> bool` returns `model.startswith("kimi-")`.
       - `send_message` raises `NotImplementedError("Moonshot backend only supports OpenAI-compatible requests; use send_openai_message")`.
       - `stream_message` raises `NotImplementedError`.
       - `send_openai_message` and `stream_openai_message` raise `NotImplementedError` with a clear message that they are implemented in the next plan.
       - `async def close(self) -> None` awaits `self._client.aclose()`.

    Keep the file minimal and focused on structure only. Do NOT implement request forwarding yet.
